API Reference

Integrate DSCLOUD into your apps with a simple REST API.

POST

/api/upload

Upload an image or video file. Returns a shareable URL and delete key.

curl -X POST https://dscloud.vercel.app/api/upload \
  -F "file=@image.png"
Response:
{
  "success": true,
  "id": "aBcDeFgH",
  "url": "https://dscloud.vercel.app/file/aBcDeFgH",
  "directlink": "https://dscloud.vercel.app/f/aBcDeFgH.jpg",
  "downloadlink": "https://dscloud.vercel.app/file/aBcDeFgH?dl"
  "deleteKey": "a1b2c3d4...",
  "originalName": "image.png",
  "size": 245760,
  "type": "image/png",
  "views": 0,
  "uploadedAt": "2026-05-14 17:18:29"
}
POST

/api/upload-url

Upload a file by providing a remote URL. The server fetches and hosts it for you.

curl -X POST https://dscloud.vercel.app/api/upload-url \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com/image.jpg"}'
Response:
{
  "success": true,
  "id": "rEmOtE42",
  "url": "https://dscloud.vercel.app/file/rEmOtE42",
  "directlink": "https://dscloud.vercel.app/f/rEmOtE42.jpg",
  "downloadlink": "https://dscloud.vercel.app/file/rEmOtE42?dl"
  "deleteKey": "35EmOtE42...",
  "originalName": "image.png",
  "size": 245760,
  "type": "image/png",
  "views": 0,
  "uploadedAt": "2026-05-14 17:18:29"
}
GET

/f/:filename

Access a file directly with a custom extension (e.g., .png). This direct link streams the resource and updates view and bandwidth stats.

curl -O https://dscloud.vercel.app/f/aBcDeFgH.png
Response:
file get download in current directory. Matches the original file's Content-Type.
GET

/api/info/:id

Read file metadata without downloading the media.

curl https://dscloud.vercel.app/api/info/aBcDeFgH
Response:
{
  "success": true,
  "id": "aBcDeFgH",
  "originalName": "image.png",
  "size": 245760,
  "type": "image/png",
  "views": 12,
  "uploadedAt": 1683886500000
}
POST

/api/delete/:id

Delete a hosted media file using the delete key returned during upload.

curl -X POST https://dscloud.vercel.app/api/delete/aBcDeFgH \
  -H "Content-Type: application/json" \
  -d '{"deleteKey":"a1b2c3d4..."}'
Response:
{
  "success": true,
  "deleted": true
}
GET

/api/stats

Fetch public service statistics and current usage limits.

curl https://dscloud.vercel.app/api/stats
Response:
{
  "success": true,
  "totalUploads": 1280,
  "totalViews": 9420,
  "bandwidth": 734003200,
  "filesStored": 840,
  "textsStored": 440,
  "limits": {
    "mediaUploadsPerHour": 50,
    "textUploadsPerHour": 100,
    "maxFileSize": "20MB",
    "supportedMedia": ["JPEG", "PNG", "GIF", "WebP", "AVIF", "MP4", "WebM"]
  },
  "botHealth": "online"
}
POST

/api/text

Host text with optional alias, admin password, view password, expiration, and formatting options.

curl -X POST https://dscloud.vercel.app/api/text \
  -H "Content-Type: application/json" \
  -d '{"text":"Hello DSCLOUD","alias":"hello","options":{"clickable":true}}'
Response:
{
  "success": true,
  "alias": "hello",
  "url": "https://dscloud.vercel.app/text.html?alias=hello",
  "rawUrl": "https://dscloud.vercel.app/t/hello",
  "deleteKey": "6a7b8c9d...",
  "expiresAt": "2026-06-12T10:00:00.000Z"
}
GET

/api/check-alias/:alias

Check whether a custom text alias is available before creating a hosted text entry.

curl https://dscloud.vercel.app/api/check-alias/hello
Response:
{
  "success": true,
  "available": true
}
GET

/t/:alias

Open the raw hosted text response for an alias.

curl https://dscloud.vercel.app/t/hello
Response:
Hello DSCLOUD
GET

/api/text/metadata/:alias

Inspect a hosted text entry before loading its content.

curl https://dscloud.vercel.app/api/text/metadata/hello
Response:
{
  "success": true,
  "alias": "hello",
  "views": 3,
  "protected": false,
  "expiresAt": "2026-06-12T10:00:00.000Z"
}
GET

/api/text/view/:alias

Fetch hosted text. Use ?pw= when the text is view-password protected.

curl "https://dscloud.vercel.app/api/text/view/hello?pw=secret"
Response:
{
  "success": true,
  "text": "Hello DSCLOUD",
  "options": {
    "preformatted": false,
    "clickable": true,
    "bbcode": false
  }
}
POST

/api/text/update/:alias

Update hosted text content, formatting options, view password, or expiration using the admin password.

curl -X POST https://dscloud.vercel.app/api/text/update/hello \
  -H "Content-Type: application/json" \
  -d '{"adminPassword":"admin-secret","text":"Updated text","options":{"preformatted":true}}'
Response:
{
  "success": true,
  "updated": true,
  "expiresAt": "2026-06-12T10:00:00.000Z"
}
POST

/api/text/delete/:alias

Delete a hosted text entry using its delete key.

curl -X POST https://dscloud.vercel.app/api/text/delete/hello \
  -H "Content-Type: application/json" \
  -d '{"deleteKey":"6a7b8c9d..."}'
Response:
{
  "success": true,
  "deleted": true
}
